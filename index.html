Nextcloud

Nextcloud est un logiciel libre d'hébergement de fichiers et une plateforme de collaboration.

Fonctionnalités

  • Synchronisation de fichiers entre ordinateurs smartphones
  • Stockage sécurisé (chiffrement des fichiers sur le serveur)
  • Partage de fichiers entre utilisateurs nextcloud ou externes
  • Serveur de fichiers WebDAV
  • Calendrier et tâches (synchro CalDAV)
  • Gestionnaire de contacts (synchro CardDAV)
  • Conférences privées audio/vidéo
  • Visionneuse de documents
  • Édition en ligne de documents
  • Galerie d'images
  • Prise de notes prenant en charge le Markdown
  • Messagerie web
  • De nombreuses applications: deck, keeweb

Clients

  • Fichiers: Navigateur web / Gestionnaires de fichier + WebDAV / Client officiel multiplateformes / Extension du gestionnaire de fichiers / Extension Thunderbird
  • Contacts: Synchronisation via le protocole CardDAV / Application "Contacts" / Thunderbird / DAVx⁵ sur Android
  • Agendas: Synchronisation via le protocole CalDAV / Application "Calendrier" / Thunderbird / DAVx⁵ sur Android
